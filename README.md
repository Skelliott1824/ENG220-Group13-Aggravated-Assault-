# Overview 
In this project we tackle the pressing issues of aggravated assault in the United States. By analyzing comprehensive datasets, we've identified key trends in affected areas and demographics. Our ultimate goal is to provide actionable data that can inform and drive the implementation of effective strategies or possible solutions to reduce aggravated assault across the country.
